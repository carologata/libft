> <h3> How to compile: </h3>

```bash
# clone repository
git clone https://github.com/carologata/42-libft.git

# enter thee folder
cd 42-libft

# compile Mandatory using Makefile
make

# compile Bonus using Makefile
make bonus

# the result of compilation is the static library
libft.a
```
